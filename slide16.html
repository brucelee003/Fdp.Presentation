Prompt Structure

Advanced Technique: Delimiters & Tags

Using structural markers to separate context from instructions for higher accuracy

Prevent Instruction Drift

Delimiters act as "guardrails," preventing the AI from confusing background information with your actual commands.

Standard Tags

Use clear markers like [START]...[END], <context>...</context>, or ### to denote sections.

Machine Readability

LLMs parse structured input more effectively, leading to better adherence to complex multi-step instructions.

prompt_editor.txt
# Use tags to isolate the source material
[CONTEXT]
Course: Microprocessors (ECE-401)
Module: Edge AI & IoT Security
Source Material: "Neural Processing Units (NPUs) optimize matrix operations at the edge, reducing latency..."
[/CONTEXT]
# Clearly separate the task instructions
[INSTRUCTIONS]
1. Act as a Senior Engineering Faculty.
2. Generate 5 multiple-choice quiz questions based strictly on the context above.
3. Tag each question with the relevant ABET Student Outcome (e.g., SO-1, SO-6).
4. Format the output as a JSON object.
[/INSTRUCTIONS]
